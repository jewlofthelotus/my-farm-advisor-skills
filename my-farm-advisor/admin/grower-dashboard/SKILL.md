# Grower Dashboard

Generate a single-file HTML dashboard for any grower/farm showing all field boundaries on a satellite basemap with interactive field filtering, GDD accumulation, and precipitation time-series.

## When to Invoke

- When you need a sharable, offline-capable visual farm dashboard
- When someone asks for "a map with GDD and rain charts for a grower"
- When presenting field-level growing-degree-day and precipitation trends
