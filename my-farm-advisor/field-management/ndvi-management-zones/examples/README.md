# NDVI Management Zones — Sample Data

This workflow does not include committed sample rasters. All runtime raster data lives under `${DATA_PIPELINE_DATA_ROOT}/data-pipeline/growers/` and is generated by the Sentinel-2 and data-pipeline imagery workflows.

## Inputs required

- A **field boundary** GeoJSON — use the field-boundaries workflow samples:
  `my-farm-advisor/field-management/field-boundaries/examples/real_10_fields_iowa.geojson`

- **Sentinel-2 NDVI rasters** — one per growing season (late June–August), produced by the Sentinel-2 imagery workflow:
  `my-farm-advisor/imagery/sentinel2-imagery/GUIDE.md`

## Expected outputs (runtime only, not committed)

All outputs land at the field level under `derived/features/`:

```
growers/<grower>/farms/<farm>/fields/<field>/derived/features/
  sentinel_scene_selection_per_year.png   # Scene ranking per year with selection highlight
  ndvi_mgmt_zones_<year>.tif              # Per-year zone raster (int16, 0/1/2, nodata=-1)
  ndvi_management_zones_k3.gpkg           # Combined GeoPackage with all years' zones
  ndvi_management_zones_per_year.png      # Multi-panel vector-based zone visualization
```

See `GUIDE.md` for the full workflow.
